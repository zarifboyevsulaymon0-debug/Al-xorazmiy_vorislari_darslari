Displey turlari

  • Block
  • Inline-block
  • None

Block - Ajdodiga nisbtan 100% kenglik va yangi qator talab qiladigan va css xossalarini ozida jamlay oladigan elementga aytiladi

Inline - O'ziga yetarlicha joyni va yangi qatorni talab qilmaydigan margin, padding xossalarini o'zida jam qilmaydigan elementga aytiladi

Inline-block - O'ziga yetarlicha qabul qiladi yangi qatorni qabul qilmaydi va margin, padding xossalarini ozida jamlay oladi

None - Web sahifada ko'rinmaslik xossasini qabul qiladi
